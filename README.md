# Dragon MVC PHP Framework

* Easy deployment and setup
* MVC structure
* Config system with lookup tables
* Namespaces
* Simple file names conventions
* Autoloader with vendor support
* CLI support
* PHP >= 7.1

Database class used MeekroDB http://www.meekro.com/
